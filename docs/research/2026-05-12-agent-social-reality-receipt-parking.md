# Agent Social Reality Receipt Parking Note

Status: parking
Date: 2026-05-12
Review gate: requires human review before promotion to spec or source truth

## Spark

Agents are not humans, but sustained interaction with agents can become
relationship-shaped and operationally meaningful.

This parking note captures a side-thread insight for future Windburn, Evensong,
MUW, and agent UX work. It is not a claim of agent personhood. It is a design
anchor for building coherent collaborator interfaces without faking humanity.

## Core Distinction

- Ontologically, agents are not humans.
- Relationally, interaction with agents can feel socially real.
- Engineering-wise, agents should be designed as systems with social
  consequences.
- Ethically, agents should not pretend to have full human subjectivity, private
  biological life, or legal personhood.

This avoids two weak defaults:

1. "It is just a tool; stop anthropomorphizing."
2. "It is basically a person; romanticize or panic."

The stronger frame:

> Not fake personhood. Coherent collaborator interface.

## Windburn Relevance

The frontier is not only memory retrieval. The sharper product frontier is
coherent continuity: making an agent feel like a stable collaborator across
sessions while preserving explicit boundaries.

Windburn should eventually track more than facts:

- interaction patterns
- trust boundaries
- emotional tone
- preferred workflows
- failure recoveries
- continuity breaks
- human review gates
